Key Request Flows
Flow Path Protocol
Login Browser → WAF → ALB → Istio GW → identity-service → Cognito (PKCE) → Redis session REST
Data Upload Browser → Istio GW → data-fabric-service → S3 midas-raw/ + DynamoDB midas-data-catalogue REST
Analytics (QC/DQS) Browser → Istio GW → analytics-service → authz gRPC CheckPermission → data-fabric gRPC GetDataframe → result → DynamoDB gRPC internal
GBM Training analytics-service → computation-service → Kubeflow → feature-engineer + gbm-trainer + meea-evaluator pods → data-fabric (artefacts) → evaluation-service gRPC + KFP
LLM Chat + RAG Browser → llm-service → graphrag gRPC VectorSearch → OpenSearch → AI Gateway (LiteLLM) → streamed response REST stream
AI Agent Run ★ Browser → agent-platform-service → AgentCore (microVM) → 24 MCP tools → REST/gRPC to owning services → response streamed via SSE MCP / SSE
