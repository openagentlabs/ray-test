## Remediation plan

2) frontend/test_project_integration.html line 56: replace entry.innerHTML with entry.textContent for the message string; for structured project data on lines 128-129 and 159-160, build DOM nodes programmatically (document.createElement) instead of setting innerHTML with JSON/template literals.

4) Add eslint-plugin-no-unsanitized to the frontend ESLint config to catch future innerHTML assignments at lint time.
5) Add a Content-Security-Policy header on the ALB or Vite dev server response to restrict inline script execution.
